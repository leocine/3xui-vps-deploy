# 系统调优

无论是否创建入站，最后都运行：

```bash
bash <(curl -sL tcp.vpsing.de)
```

首次运行会安装到 `/usr/local/bin/tcp.sh`，并创建快捷命令 `t`。

不要提示用户选择，进入交互菜单后自动依次输入：

```text
1 回车：设置 IPv4 优先解析
2 回车：开启 BBR + FQ
3 回车：生产级内核调优
4 回车：网卡多队列均衡
0 回车：退出脚本
```

每一步提示“按回车返回”时自动回车继续。

不要执行：

```text
5 回退
6 更新
7 卸载
```

除非用户明确要求。

检查：

- BBR 是否开启
- IPv4 优先是否开启
- x-ui 是否 active
- 面板 HTTPS 是否可访问
