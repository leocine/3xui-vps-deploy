# 最终输出与订阅获取

最后输出：

- 面板 URL
- 面板用户名
- 面板密码
- 证书路径
- IPv6 是否关闭
- IPv4 优先状态
- BBR 状态
- Clash/Mihomo 订阅是否开启，路径 `/clash/`
- 如果创建了入站，列出 4 个入站端口和客户端信息

## 获取订阅

告诉用户：

1. 访问 3x-ui 面板 URL 并登录。
2. 进入“客户端”菜单。
3. 在列表里找到 `admin`。
4. 点击 `admin` 的二维码。
5. 复制订阅信息。
6. 导入 VPN 客户端，例如小火箭、Clash Verge、v2rayN。
7. 重点提醒 Clash Verge 用户：不要把第 5 步复制出来的订阅链接直接导入 Clash Verge。必须先把这个链接放到浏览器打开，然后在打开的页面里选中“Clash 订阅信息”并复制，最后再导入 Clash Verge。这个步骤很容易搞错，必须单独强调。

最后提醒用户立刻修改 root 密码。
